# Deutscher Baukasten

Deutscher Baukasten ist eine Sammlung von kostenlosen Online-Rechnern und Tools für Alltag, Arbeit und Finanzen. Diese Website stellt einfache Rechner bereit, die direkt im Browser funktionieren.

## Struktur
- `index.html` – Startseite mit Links zu allen Rechnern.
- `styles.css` – zentrales Stylesheet.
- `impressum.html` – Impressum gemäß § 5 TMG.
- `datenschutz.html` – Datenschutzerklärung gemäß DSGVO.

## Lokale Nutzung
Öffnen Sie `index.html` in Ihrem Browser, um die Toolübersicht zu sehen und die Rechner zu verwenden. Die Dateien benötigen keinen Server und funktionieren offline im Browser.
